Aunque esta versión no es tan popular como la versión de java y la cantidad de streamers y servidores sean inferiores es una versión mucho mas elaborada puesto que posee muchas características que la versión de Java no tienes además de ser una versión optimizada para que pueda funcionar con 2GB de memoria RAM.
