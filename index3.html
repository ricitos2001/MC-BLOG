Minecraft java

Minecraft java es una versión disponible para ordenadores y portátiles para las Microsoft Windows, Mac-Os y Linux. Aunque inusualmente podemos encontrar esta versión en dispositivos móviles. Esta versión es popularmente conocida gracias a la enorme cantidad de streamers que realizan gameplays en redes sociales como YouTube o Twitch, por la gran variedad de servidores a los que puedes unirte y por la gran variedad de mods que puedes añadir en el juego. Esta versión se considera como la versión mas pesada y desde la versión 1.21 es recomendable usar 4GB de memoria RAM como mínimo para evitar que se generen problemas a la hora de crear o cargar el mundo o para evitar el lag o tener crasheos del juego.

Minecraft bedrock

Minecraft bedrock es una versión disponible para todas las plataformas excepto para las distribuciones de Linux. Aunque esta versión no es tan popular como la versión de java y la cantidad de streamers y servidores sean inferiores es una versión mucho mas elaborada puesto que posee muchas características que la versión de Java no tienes además de ser una versión optimizada para que pueda funcionar con 2GB de memoria RAM.

Minecraft PE

Minecraft PE es la versión original de Minecraft actualmente disponible para algunas videoconsolas como la Wii U, la PS3 o la PS4 y siendo esta la versión menos conocida. En esta versión no se pueden acceder a servidores como en las versiones de Java o bedrock y no posee la característica del uso de comandos en los y chat de texto pero si que se pueden crear salas de minijuegos y realizar partidas multiplicador lo cual es una característica que no existen en las versiones de Java y Bedrock. Esta versión es mucho mas sencilla y a diferencia de las dos versiones anteriores el almacenamiento y la memoria a utilizar son muy inferiores por lo que se puede disfrutar de este juego sin tener problemas de lag o crasheos.
